Задание 1. Написать программу "Проверяем скорость Интернета с помощью Python" (0,5 б)
Как известно: модуль speedtest-cli представляет собой интерфейс командной строки для тестирования пропускной способности интернета с использованием сервиса speedtest.net.
Устанавливаем модуль:
pip install speedtest-cli
Далее импортируем модуль и вводим команды для использования функционала данного модуля:
import speedtest-cli
test = speedtest.Speedtest()
# скорость скачивания
download = test.download()
print(download)
# скорость загрузки
upload = test.upload()
print(upload)
Команды download() и upload() выдают соответственно результат измерения скорости скачивания и загрузки данных.
Весь результат модуль отдает в байтах, поэтому для наглядности нужно полученный результат переводить в мегабайты.

Задание 8. Написать программу "Скачиваем видео с YouTube". Существует малоизвестный модуль pytube, который позволяет отфильтровывать и скачивать видео с платформы YouTube. Рекомендую разобраться, как скачать видео с популярного видеохостинга, используя Python. Доп. баллы назначаются, если программа позволит скачать видео с RuTube.


Устанавливаем модуль pytube:
python -m pip install pytube
Далее импортируем его и используем функции:
 import pytube
# создаем объект yt
 yt = pytube.YouTube('ссылка на видео')
# отфильтровываем поток
 vd = yt.streams.filter(progressive=True, file_extension='mp4')
# загружаем видео
 vd.download()
